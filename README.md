# Project Photon

Babylon.jsで軽量かつ操作が容易な3Dエディタを作ろうとしているプロジェクトです。
名前は変わるかも


## 使い方 (一般)
1. デプロイ先のエディタにアクセス [リンク](https://photon-seven.vercel.app/editor/EditorPage)


## 使い方 (開発)
1. このリポジトリをクローンする
`$git clone https://github.com/Imanect0/3D-Editor.git`
2. `3D-Editor`に入る
`cd 3D-Editor`
3. `yarn`と打つ
4. `yarn dev`と打つ
5. http://localhost:3000/ にアクセスする
